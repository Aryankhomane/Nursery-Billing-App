# Nursery Billing App

A Nursery Billing and Payment Tracking Application built using AppSheet and Google Sheets.

## Features

* Customer Management
* Plant Billing System
* Automatic Bill Generation
* PDF Invoice Creation
* Pending Payment Tracking
* Payment Status (Paid / Pending)
* WhatsApp Payment Reminder Button
* Customer Phone Number Management

## Technologies Used

* AppSheet
* Google Sheets
* Google Drive
* WhatsApp Integration

## Project Workflow

1. Create a Customer
2. Add Plants to Bill
3. Generate Invoice
4. Track Payments
5. Send WhatsApp Reminders for Pending Payments

## Future Improvements

* Inventory Management
* Sales Dashboard
* Due Date Tracking
* Customer Payment History
* Expense Tracking
